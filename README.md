# Besucherverwaltung

Dieses ist zunächst ein Prototyp von Otto's VBA-Anwendung als webbasierte Ausführung.  
Es können in dieser Version nur Veranstaltungen, Berater, Besucher verwaltet werden.

## Wichtig

### Diese Version ist für Desktops optimiert. Für Tablets oder Smartphone Displays müssen bei Bedarf Anpassungen erfolgen

## PCafe Bersucherverwaltung – Funktionen

### Registrierung der Besucher

*   Name
*   Spende
*   Dauer der Beratung
*   Besuchte Veranstaltung
*   Thema der Beratung \*
*   Benutztes Gerät
*   Datum \*
*   Ort der Beratung und Art (Beratung oder Info-Veranst. Mit Thema) \*

### Registrierung der Beratungen/Veranstaltungen

*   Datum
*   Ort der Beratung
*   Ort und Art der Beratung
*   Thema der Info-Veranst
*   Gesamt Spenden
*   Anwesende Berater

### Berater

*   Name
*   Telefonnummer
*   E-Mail
*   Durchgeführte Beratungen/Veranstaltungen \*

### Besucher

*   Name
*   Anrede
*   Telefon
*   E-Mail
*   Aktiv/inaktiv
*   Newsletter ja/nein
*   Anschrift (optional)

### Auswertungen

*   Anzahl und Details der Besuche
*   Anzahl und Details der Veranstaltungen

### Sonstige Funktionen

*   Sammel-Email an alle aktiven Besucher
*   Sammel-Email an alle Newsletterempfänger
*   Sammel-Email an alle Berater
*   Sammel-Email an alle Besucher einer Veranstaltung

### Verwaltungsfunktionen

*   Stammdaten
*   Datensicherung

\* diese Daten werden über den Link zur Veranstaltung erfasst
